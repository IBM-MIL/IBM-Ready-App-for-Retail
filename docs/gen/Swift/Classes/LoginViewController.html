  • loginButtonAction(_:)

    This method is called when the loginButton is pressed. This method checks to see if the username and password entered in the textfields are of valid syntax. It does this by calling the checkEmail and checkPassword methods of the utility class SyntaxChecker.swift. This class assumes that the following regular expression is allowed for an email [A-Z0-9a-z.%+-]+@[A-Za-z0-9.-]+.[A-Za-z]{2,4} and for a password [A-Z0-9a-z@!&%$#*?]+. This method also calls the login method
